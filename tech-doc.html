Introduction

What is HTML?

HTML stands for Hyper Text Markup Language. It is the standard markup language for creating web pages, and describes the structure of a web page. 'HyperText' refers to links that connect web pages to one another, either within a single website or between websites.

HTML uses 'markup' to annotate text, images, and other content for display in a Web browser.

Example Code:

<!DOCTYPE html> <html> <head> <title>Page Title</title> </head> <body> </body> </html>

What is an HTML Element?

An HTML element is defined by a start tag, some content, and an end tag. The elements tell the browser how the content should be displayed.

Anatomy of an HTML Element:

<tagname>Content</tagname>

The HTML element is everything from the start tag to the end tag.

The name of an element within a tag is case-insensitive. It can be written in uppercase, lowercase, or a mixture.

The Main Parts of an HTML Element:

  • Opening Tag: Consists of the name of the element
  • Closing Tag: Same as the opening tag, but it includes a forward slash '(/)' before the element name
  • Content: Content of the element; typically just text
  • Element: Opening tag, Closing tag, and the content makeup an element

HTML

Anatmoy of an HTML document

<!DOCTYPE html> <html> <head> <meta charset="utf-8">
<meta name="viewport" content="width=device-width">
<title>TestPage</title>
</head>
<body>
<img src="images/cat-icon.jpg" alt="Test Image"/>
</body>
</html>

Above Code Explanation:

  • !doctype html - tells the browser what to expect

  • html - wraps all the content on the entire page; sometimes known as the root element.

  • head - acts as a container for everything the viewer doesn't need to see.

  • meta charset="utf-8" - sets the chracter the website should be used; includes most characters from the vast majority of written languages.

  • meta name="viewport" content="width=device-width" - ensures the page renders at the width of the viewport; preventing pages from being too wide or shrinking down on mobile browsers.

  • title - sets the title of the web page.

  • body - contains all the content that users will see on the web page. Including: text, images, videos, games, etc.

A few examples:

  • <h1>
  • represents the main heading/subject for the whole page.
  • <p>
  • defines a paragraph.
  • <strong>
  • allows content (text) within element to be bold.
